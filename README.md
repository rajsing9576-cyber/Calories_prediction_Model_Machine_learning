Calories Burnt Prediction System

A Machine Learning project that predicts the number of calories burned based on user physical and workout data. This project uses supervised learning algorithms to estimate calories burned during exercise sessions.

📌 Project Overview

The Calories Burnt Prediction System is designed to predict the calories burned by a person using various health and activity-related parameters such as:

Gender
Age
Height
Weight
Exercise Duration
Heart Rate
Body Temperature

The model is trained on historical workout data and can help users estimate calorie expenditure accurately.
